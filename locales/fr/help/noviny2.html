Personnalisation de la page d'accueil

Zone centrale

La page d'accueil du thème affiche, sur fond bleu, le dernier billet de la première catégorie du blog (zone de code nommé "select"), puis en-dessous le dernier billet de chacune des autres catégories (zone de code nommé "overview").

Colonne de droite
Les "Dernières nouvelles" listent les 4 derniers billets sans catégorie.

Vous pouvez modifier les codes dans chacune des zones de code, afficher l'un et pas l'autre.
Au cas où vous auriez besoin de retrouver les codes initiaux il suffit de consulter le fichier readme.md situé en racine du dossier du thème (via FTP).

Chacune de ces zones de code possède sa propre css située dans le dossier /css du thème.
