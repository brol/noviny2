Customizing the home page

Central Zone

The theme of the home page displays, blue background, the last ticket the first blog category (area code named "select") and below the last ticket to each of the other categories (area code named "overview").

Right Column
The "Breaking news" list the Notes without category.

You can change the codes in each code areas, display one and not the other.
If you need to locate the original code simply visit the readme.md file located in the root theme folder (via FTP).

Each of these areas of code has its own css css located in the theme folder.
